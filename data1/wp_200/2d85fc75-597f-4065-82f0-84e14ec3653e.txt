GML est l'abréviation de noms de langages informatiques de description de données : Geography Markup Language; Graph Modelling Language; Generalized Markup Language (ou Goldfarb, Mosher et Lorie) - voir aussi SGML.
GML désigne également un langage de programmation: Game Maker Language.
GML peut désigner un sigle d'entreprise : Group Menatep Limited.
GML est le code AITA de l'aéroport Gostomel à Kiev, Ukraine.
GML est l'acronyme de Glycerol Mono-Laurate, un composé naturel à action bactéricide.
